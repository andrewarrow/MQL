NAME:
   alamode - alamode command line interface for mode

USAGE:
   alamode [global options] command [command options] [arguments...]

VERSION:
   16

COMMANDS:
     spaces, p   spaces
     reports, r  reports
     queries, q  queries
     sql, s      sql
     run, u      run
     help, h     Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --help, -h     show help
   --version, -v  print the version
